# Amazon UI Clone

A front-end clone of Amazon's e-commerce website built using HTML and CSS,
replicating the core user interface and layout of the original platform

## 📌 About the Project
This project is a pixel-perfect UI clone of Amazon's web interface,
built to strengthen front-end development skills. The focus is on
replicating the visual design and layout using only HTML and CSS.
JavaScript logic for interactivity is currently being added.
